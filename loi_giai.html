Tìm tọa độ giao điểm của đồ thị hàm số \(f(x)=x^{3} - 3 x^{2} + 2\) với đường thẳng \(f(x)=- 2 x + 2\)

Ta có phương trình hoành độ giao điểm :
\(x^{3} - 3 x^{2} + 2 = - 2 x + 2\)

\(x^{3} - 3 x^{2} + 2 = - 2 x + 2\)
\(\Leftrightarrow x^{3} - 3 x^{2} + 2 x = 0\)
\(\Leftrightarrow x \left(x - 2\right) \left(x - 1\right) = 0\)
\(\Leftrightarrow x=\left [ 0, \quad 1, \quad 2\right ]\)

Từ các nghiệm của phương trình hoành độ giao điểm ta xác định giao điểm (\(x\),\(f(x)\))
\(\begin{cases}\left ( 0, \quad 2\right ) \\ \left ( 1, \quad 0\right ) \\ \left ( 2, \quad -2\right ) \\ \end{cases}\)
