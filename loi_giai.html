Khảo sát hàm số : \(f(x) = -4 + 3 x^{2} + x^{3}\)

Hàm số là hàm đa thức nên luôn xác định trên tập số thực
\(D=\mathbb{R}\)


\(f'(x)=(-4 + 3 x^{2} + x^{3})'\)
\(\Leftrightarrow \operatorname{f'}{\left (x \right )} = 6 x + 3 x^{2}\)

\(6 x + 3 x^{2} = 0\)
\(\Leftrightarrow 3 x \left(2 + x\right) = 0\)
\(\Leftrightarrow x=\begin{cases}-2 \\ 0 \\ \end{cases}\)

\(lim_{x\to-\infty}-4 + 3 x^{2} + x^{3}=-\infty\)
\(lim_{x\to\infty}-4 + 3 x^{2} + x^{3}=\infty\)


Nhận xét :
Hàm số đạt cực tiểu tại điểm : \(\left ( 0, \quad -4\right )\)
Hàm số đạt cực đại tại điểm : \(\left ( -2, \quad 0\right )\)
Hàm số có điểm uốn : \(\left ( -1, \quad -2\right )\)
